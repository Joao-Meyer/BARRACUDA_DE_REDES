Intalar o pacote nfs-kernel-server

~ apt install nfs-kernel-server

Criar um arquivo dentro do diretório /opt

~ mkdir /opt/docs 

Modificar as permissões do arquivo

~ chmod 777 /opt/docs (o arquivo pode ter qualquer nome)

Entrar no arquivo de configuração do NFS

~ vi /etc/exports 

Usar o seguinte esquema para a identificar o sistema de compartilhamento:

Quem vou comp.?     		Para quem vou comp.?não tem espaço				Forma? 
    /opt/docs          (aqui colocamos o IP da rede e máscara CIDR)		(aqui colocamos será "ro"somente leitura
										 ou se ele será "rw" leitura e escrita)
Como ficará: 

# Diretório Compartilhado 

/opt/docs       192.168.0.0     (rw,sync,no_subtree_check)

Reinicie o serviço 

~ /etc/init.d/nfs-kernel-server restart

Para verificar monte o repositório (em um client)
 
Verifique se os arquivos foram mesmo compartilhados

~ showmount -e 192.168.0.1 (ip do servidor desejado)

Crie uma pasta no /mnt para os arquivos

~ mkdir /mnt/gustavo

Monte o repositório 

mount -t nfs -o nfsvers=3 192.168.0.1:/opt/docs /mnt/gustavo

Entre na pasta e verifique
