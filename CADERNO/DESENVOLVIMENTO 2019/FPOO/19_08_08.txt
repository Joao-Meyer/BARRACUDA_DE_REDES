SALVANDO O ARQUIVO

Para que o programa seja salvo corretamente é necessário que o nome do arquivo seja o mesmo da classe, no caso, como utilizamos o bloco de notas são necessários alguns cuidados a mais.

A classe do programa é "Teste", então salvei como "Teste.java", além disso, foi necessário mudar a extensão do arquivo para todos e colocar o nome do arquivo ente aspas para que o bloco de notas preservasse a extensão ".java" do arquivo.
