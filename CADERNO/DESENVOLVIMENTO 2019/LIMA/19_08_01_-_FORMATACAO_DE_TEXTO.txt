Passos para a criação de um site para um cliente: (Recomendados pelo professor)

1º - Domínio: É o nome pelo qual o seu site ficará conhecido na internet (Verificar se o cliente tem ou não, se há disponibinibilidade, etc)
	|-> Registro.br
		|-> +- R$40,00/ano
	|-> Registrado no nome do cliente (Informar para ele que é necessário renovar a cada x tempo e fazer a conta e registrar o domínio no nome do cliente)

2º - Hospedagem: Comprar/Alugar um servidor onde será alocado seu site
	|-> locaweb.com.br
	|-> uolhost.com.br
	|-> etc...
		|-> Por volta de R$30,00 - R$40,00 /mes

HTML (HIPER TEXT MARKUP LANGUAGE)
	|-> Boas práticas
		|-> Identar o código

		|-> Comentar o código ( <!-- --> )

	|-> Alguns comandos
		|-> <p> - serve para colocar um parágrafo com texto
			
		|-> <p align =	"left"
				"center"
				"right"
				"justify"
		|-> <b> - Negrito para qualquer palavra não relevante
			
		|-> <strong> - Negrito para palavras relevantes para o buscador
			
		|-> <i> - Italico
			
		|-> <u> - Sublinhado
			
		|-> <small> - Realça a letra em tamanho pequeno
			
		|-> <big> - Realça a letra em tamanho maior
			
		|-> <del> - Risca uma palavra
			
		|-> <sup> - Coloca o texto elevado ao lado do outro
			Ex: 2 elevado a 3
			
		|-> <sub> - Coloca o texto abaixo ao lado do outro
			Ex: Formula do H2O	

		|-> <br> - Quebra de linha (como um enter) (não precisa ser fechado)

		|-> <meta charset="utf-8"> - Corrige o padrão de acentuação (PT-BR) para o navegador
			
		|-> <h1> ... <h6> - servem para colocar títulos no site (h1 sendo o título principal, h2 subtítulo e etc...)
			
		|-> <font	face=" " - Altera o tipo da letra
				
				size=" " - Altera o tamanho da letra
					
				color=" " - Altera a cor da letra
			
		|-> Color piker: site para achar as cores
		
		|-> <center> - centraliza o conteúdo
