#################SAMBA########################

~ apt install libcups2 

~ apt install samba 

~ apt install samba-common


*CRIAR UM DIRETÓRIO ANÔNIMO* 

~ mkdir /srv/share/anonimo -p 

*MUDAR O NOME DO GRUPO* 

~ chown -R root:users /srv/share/anonimo

*MUDAR PERMISSÕES* 

~ chmod g+w 

################CONFIGURAR O SAMBA##############

~ mv /etc/samba/smb.conf /etc/samba/smb.conf.bkp

~ vi /etc/samba/smb.conf 

[global]
workgroup= GRUPO 
security= user
map to guest= bad user

[anonimo]
patch= /srv/share/anonimo 
force group= users
browsable= yes
writable= yes
guest ok= yes

reiniciar o serviço


#####################SAMBA USUÁRIO AUTENTICADO###############

~ vi /srv/share/documentos -p

~ chmod 777 /srv/share/documentos

~ chgrp users /srv/share/documentos

~ vi /etc/samba/smb.conf 

[global]
workgroup= GRUPO 
security= user
map to guest= bad user

[documentos] 
comment= compartilhamento autenticado 
path= /srv/share/documentos
valid users= @users
force group= users 
writable= yes 

reiniciar o serviço 

adicionar o usuário "tux"

~ useradd tux -m -G users
~ passwd tux 
~ smbpasswd -a 

adicionar usuário no samba 

~ addgroup (usuário desejado) users 
~ smbpasswd (senha desejada)


###########################SAMBA HOME DO USUÁRIO#################

Fazer uma nova sessão em /etc/samba/smb.conf

[homes]
comment= Home do usuário
browseable= no
valid users= %S
writable= yes
create mask= 0700
directory mask= 0700 

Salvar o arquivo e reiniciar o serviço em seguida.
